# RoboSpeakerPy

RoboSpeakerPy is a simple Python text-to-speech project created by Yadvendra.  
It takes user input and speaks it using the Windows SAPI5 voice engine.

## Features

- Text-to-speech conversion
- User input based speech
- Uses pyttsx3
- Beginner-friendly Python project

## Tech Stack

- Python
- pyttsx3
- pywin32
- Windows SAPI5

## How to Run

```bash
pip install pyttsx3 pywin32
python main.py
